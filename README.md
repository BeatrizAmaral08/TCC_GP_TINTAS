# GPTintas

Sistema de gerenciamento e venda de tintas desenvolvido para controle de produtos, pedidos e usuários.

---

# Objetivo

O GPTintas tem como objetivo facilitar o gerenciamento de uma loja de tintas, permitindo controle de estoque, gerenciamento de produtos e
realização de compras online.

O sistema possui diferentes níveis de acesso para garantir segurança e organização.

---

# Perfis de Usuário

## Comprador

* Visualizar produtos
* Adicionar produtos ao carrinho
* Realizar compras
* Visualizar pedidos

## Repositor

* Adicionar produtos
* Editar produtos
* Excluir produtos
* Atualizar estoque

## Dev

* Controle total do sistema
* Gerenciamento de usuários
* Gerenciamento do site
* Controle de permissões
